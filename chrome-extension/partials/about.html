About this project

This project, called "Steam Backlog", is an attempt of providing easier and better way to browse our always growing steam libraries while helping users decide what to play next.

Are going to be any updates? Some new features and improvements to the current project are planned for the future. My Steam account is very valuable to me. I'm worried about my privacy. This extension uses the public Steam API to get all the information, and cannot access any private data on your account or computer. Also, all the code is hosted on Github and everybody is free to see it (and even propose changes).
